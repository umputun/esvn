Editing

To edit a local copy of a file use the Edit option ( Modify->Edit (F4) ). The selected file is opened in the default editor specified in the Editor option. To define the default editor, select File->Options General+Programs->Editor.

Double-click the file to open it in the default editor specified in the Editor field of the Options dialog.
