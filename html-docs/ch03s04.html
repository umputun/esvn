Browsing Remote Repositories

Using the Repository Browser it is possible to browse the structure and contents of a remote repository without performing a checkout to the local machine ( File->Browse repository ... (Ctrl+B) ).

To browse a remote repository enter the URL and click the Browse button. After a few seconds the tree and file views will be populated with the structure and file name resources stored in the remote repository. The Repository Browser also enables checkout and view operations to be executed without exiting.

A quick method to insert the URL string is to select the root folder of the working copy, select File->Browse repository ... (Ctrl+B) , then click the Browse button.
