This chapter provides explanation to the various options available in eSvn. The operations are sorted in accordance with the eSvn menu structure.

Opening a Working Directory

If you already have a local copy of a subversion repository on your machine, it can be opened with eSvn. It is also possible to open other file systems not under subversion control as a prelude to importing to the server (the section called “Importing a File System”).

Procedure 3.1. Opening a working directory

  1. From the main menu select File->Open working directory ... (Ctrl+O) . The Open dialog is displayed.

  2. Change path to the root folder of the workspace directory, then click OK. The workspace is opened in eSvn
