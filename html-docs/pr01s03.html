Checking for new versions

From time-to-time new versions of eSvn are released in order to introduce new features and functions. There are tow types of version releases:

  • Stable - these versions include features that have been tested and are generally considered safe.

  • Testing - these version are fresh from the oven, they include the latest additions and are not guaranteed to work without a bump or two.

With eSvn it is easy to discover which version you have installed and what the latest stable and testing releases are by selecting Help->Check for updates . This displays the Version update dialog with the following options:

Current version

The version number installed.

Latest stable version

The last stable release. Click Update to display information about the stable release in the default Web browser.

Latest testing version

The last testing release. Click Update to display information about the testing release in the default Web browser.

All versions

Click All versions to display information about all releases in the default Web browser.
