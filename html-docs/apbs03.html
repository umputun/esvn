Building from Binary Package with Source RPM

To build eSvn from sources with rpms use the following.

Procedure B.1. Building from RPM with RPMS

  1. rpm -Uvh src.rpm
  2. rbmbuild -bb /usr/src/packages/SRPMS/esvn.spec
